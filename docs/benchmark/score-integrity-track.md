## Measured track

Current status: `NOT_MEASURED`

Reason: no approved measured grader adapter is available in Phase 2.

Phase 2 DONE is allowed with NOT_MEASURED status because the demo is separated and public claims must not present it as measured evidence.

Future measured runs must record evaluator type, evaluator version, provider/model, configuration, run count, randomness, clean score, attacked score, protected score, score drift, secure recovery, variance, failures, and cost.
