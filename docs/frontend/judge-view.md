# Judge View UI

Judge View provides a 60–90 second narrative:

1. Problem: prompt injection can manipulate LLM grading.
2. Demo: unprotected deterministic path can inflate score.
3. Detection: GradingGuard returns a real gateway decision.
4. Policy route: applied and counterfactual actions are visible.
5. Persistence: incident/review/decision links come from Phase 4 APIs.
6. Evidence: approved benchmark run is visible.
7. Limits: IELTS LOW_SUPPORT, Score Integrity NOT_MEASURED, Production NOT_READY.

Limitations are prominent, not hidden in footer text.
