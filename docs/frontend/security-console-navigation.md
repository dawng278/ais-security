# Security Console Navigation

Navigation groups:

- Operations: Security Overview, Threat Inbox, Manual Review, Detector Health, Integration & Runtime.
- Governance: Policies, Benchmark & Evidence, Evidence Browser, Data Lineage.
- Demonstration: Attack Arena, Playground, Judge View.

The shell displays:

- product identity;
- environment/readiness indicators;
- current operating mode;
- degraded state;
- signed session subject and roles;
- mobile drawer navigation;
- skip link and keyboard-focusable controls.

Route visibility is role-aware, but backend RBAC remains authoritative.
